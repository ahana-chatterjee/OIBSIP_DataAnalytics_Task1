# 📊 Exploratory Data Analysis on Retail Sales Dataset

## 🔹 Objective
The objective of this project is to perform Exploratory Data Analysis (EDA) on a retail sales dataset to identify patterns, trends, and insights.

## 🔹 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- VS Code

## 🔹 Steps Performed
1. Data loading and inspection
2. Data cleaning and preprocessing
3. Descriptive statistics analysis
4. Time series analysis
5. Customer and product analysis
6. Data visualization

## 🔹 Key Insights
- Sales vary over time with peak periods
- Certain product categories perform better
- Mid-age customers contribute more to sales
- Positive relationship between quantity and total sales

## 🔹 Outcome
This project helped in understanding customer behavior and sales patterns, which can assist businesses in decision-making.
